# 🎨 Ndiso Designs: 24/7 Art Stream

Welcome to the official repository for the **Ndiso Designs** 24/7 YouTube stream. This project broadcasts a continuous loop of African-inspired patterns, public murals, and custom apparel designs.

## ✨ About the Artist
**Ntsindiso Nq** is a multidisciplinary artist based in Verulam, KZN. 
* **Specialties:** Surface Pattern Design, Murals, and Apparel.
* **Recognition:** Design Indaba Emerging Creative (2015).
* **Current Pursuit:** UNISA Student & scaling international Print-on-Demand sales.

## 🎵 Soundtrack
